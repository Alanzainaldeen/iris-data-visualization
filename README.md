# 🌸 Iris Data Visualization Project

A simple yet powerful data visualization project using Python to explore the famous Iris dataset. This project demonstrates how to generate random data, visualize relationships, and encode categorical variables for better insights.

---

## 📌 Project Overview

This project focuses on:

* Understanding the Iris dataset
* Creating scatter plots using Matplotlib
* Using colors and sizes to represent additional dimensions
* Handling categorical data (species) for visualization

---

## 📊 Features

* 📈 Scatter plot visualization
* 🎨 Color mapping based on species
* 🔍 Adjustable point sizes and transparency
* 🧪 Random data generation using NumPy

---

## 🛠️ Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn (for encoding labels)

---

## 📁 Dataset

The project uses the classic Iris dataset, which contains:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width
* Species (Setosa, Versicolor, Virginica)
